# Automação de Formulário com Python

Este projeto demonstra como automatizar o preenchimento de formulários usando Python e a biblioteca PyAutoGUI.
É um exemplo prático de automação para tarefas repetitivas no computador.

# Funcionalidades
- Abrir o navegador Chrome
- Acessar um site específico
- Preencher campos de e-mail e telefone
- Enviar o formulário automaticamente

# Tecnologias utilizadas
- Python
- PyAutoGUI
- time
